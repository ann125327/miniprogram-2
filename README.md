# 记账喵小程序

## 项目介绍
本项目为记账小程序，基于HTML、CSS、JavaScript等技术，实现了基本的记账、统计收支、控制预算以及用户登录的功能。本项目旨在帮助用户记录收支明细、按照分类查看收支，并进行数据可视化，用户可随时设置预算，帮助其养成节约的习惯，避免不知不觉超出预算。项目界面风格简约，便于用户查看收支明细，登录功能丰富了用户的使用体验。

## 项目运行
由于小程序正式版的发布需要备案和审批，我们的项目目前是小程序（体验版），体验版小程序无法搜索和分享，只有小程序项目管理员手动输入微信号添加的指定用户可以通过扫码查看（如果扫码无法打开请飞书联系【何舒平】）

### 安装微信开发者工具
点击以下链接：  
[微信开发者工具下载](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)  

安装稳定版，并通过微信注册开发者账号（或选择测试号），获取小程序的appid。

### 克隆项目到本地
```bash
git clone https://github.com/ann125327/miniprogram-2.git
```

### 导入项目并运行
导入项目后，在`project.config.json`文件里将appid修改为刚刚获取的自己的小程序id，后点击编译预览即可运行项目。
